# メタディスクリプション雛形テンプレート

作成日: 2026-05-02
対象: led-proshop.com 記事・カテゴリページ
用途: 各記事のmeta descriptionの改訂・新規作成時の参照雛形
制約: 155文字以内・キーワード先頭・CTA含む

---

## 案A — 省エネ訴求型

**方向性**: 電気代削減・2027年問題（蛍光灯生産終了）・投資回収年数を先頭に打ち出す。
コスト意識の高い施設管理者・経営者層に刺さる訴求。

```
【用途業種キーワード】のLED照明なら蛍光灯比最大60%コスト削減。
2027年蛍光灯廃止前に補助金活用でゼロリスク移行。
年間削減額シミュレーション・施工事例を無料でご確認ください。
```

文字数目安: 約100〜120文字（業種キーワードで調整）

**穴埋め変数**:
- `【用途業種キーワード】` → 例: 飲食店、工場・倉庫、クリニック 等
- 省エネ率は実績値に合わせて変更（49%〜65%の範囲で実績ベース）

**適用先記事種別**: 工場・倉庫・オフィス・クリニック・学校系（コスト重視）

---

## 案B — 品質訴求型

**方向性**: Ra（演色性）・色温度・フリッカーフリーといった照明品質スペックを先頭に。
内装デザイナー・建築士・照明設計の専門家層、品質重視の飲食店オーナーに刺さる訴求。

```
Ra90以上の高演色LEDで【色や質感が重要な業種】の照明品質を革新。
2027年問題対応済み・フリッカーフリー・IP65対応まで業種別に最適設計。
Ra別比較表・選定チャートを今すぐ確認。
```

文字数目安: 約110〜135文字（業種キーワードで調整）

**穴埋め変数**:
- `【色や質感が重要な業種】` → 例: ジュエリーショールーム、精肉・鮮魚店、美容室
- Ra値は記事の推奨スペックに合わせて記載（Ra90 / Ra95 / Ra98 等）

**適用先記事種別**: 小売店・美容・飲食（高級志向）・ジュエリー・精肉鮮魚・ギャラリー系

---

## 案C — 専門性訴求型

**方向性**: 「業種専門のLED設計ノウハウ」「施工実績○○件」「B2B専門商社」という権威性を前面に。
複数拠点を持つ法人・FC本部・ゼネコン・設備会社など、信頼性で選ぶ顧客層向け。

```
【業種】専門のLED設計ノウハウ・業種別施工事例を公開。
電気工事会社・内装施工会社・FC本部の一括導入をワンストップ対応。
工場直送価格で全国納品・業種別LEDセレクションはこちら。
```

文字数目安: 約105〜130文字（業種キーワードで調整）

**穴埋め変数**:
- `【業種】` → 例: 飲食チェーン、医療・クリニック、ホテル・宿泊施設
- 「一括導入」「ワンストップ」は法人向けの言葉として維持推奨

**適用先記事種別**: ホテル・チェーン飲食・物流・大型施設・法人一括発注向けカテゴリページ

---

## 使い分けガイド

| ページ種別 | 推奨案 | 理由 |
|---|---|---|
| 施工事例（工場・倉庫・オフィス） | 案A | コスト削減数値が実績として示せる |
| 施工事例（美容・ジュエリー・飲食高級） | 案B | Ra・色温度の専門知識が差別化になる |
| カテゴリまとめページ / ピラーページ | 案C | 専門性と一括対応力でB2B信頼を構築 |
| ガイド記事（LED選び方・比較記事） | 案B | 技術用語先頭でサーチインテント一致 |
| 施工事例（チェーン・ホテル・学校） | 案A or C | 規模感と実績数値の組み合わせ |

---

## 共通ルール

1. **155文字以内** を必ず守る（Google検索結果での表示上限の目安）
2. **キーワードは先頭** に配置（業種名・省エネ率・Ra値 等）
3. **CTA（行動喚起）** を末尾に必ず含む（「確認ください」「こちら」「今すぐ」等）
4. 数値は実記事の実績値を使用する（架空数値禁止）
5. 「業界最安値」「No.1」等の最上級表現は使用しない（景品表示法考慮）
6. 本番HTMLのmeta descriptionへの反映はproshop-director承認後に実施
