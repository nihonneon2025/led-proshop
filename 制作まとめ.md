## ⏱ 制作時間（概算）

| 作業内容 | 時間 |
|---|---|
| サイトデザイン・HTML/CSS/JS構築 | 約8〜12時間 |
| フォーム・メール機能実装（PHP） | 約3時間 |
| Stripe決済統合 | 約2時間 |

## 🛠 使用ツール

| ツール | 用途 |
|---|---|
| GitHub Pages | サイトホスティング（led-proshop.com） |
| XServer | PHPメールスクリプト置き場（nihon-neon.jp） |
| Stripe | クレジットカード・Apple Pay決済 |
| Claude（Cowork） | コーディング・実装・デバッグ全般 |
| HTML / CSS / JavaScript | フロントエンド構築 |
| PHP | メール送信・Stripe API連携 |

## 💰 予算（月額・年額）

| 項目 | 費用 |
|---|---|
| GitHub Pages | 無料 |
| XServer（nihon-neon.jp） | 既存契約のため追加費用なし |
| ドメイン led-proshop.com | 約1,500〜2,000円/年 |
| Stripe 決済手数料 | 売上の3.6%（売れた分だけ） |
| Claude Cowork | サブスクリプション費用のみ |

初期費用はほぼゼロ、ランニングコストも最小限で構築。

---

## 📋 実装した主な機能

- 商品カタログ表示（カテゴリ別・検索・フィルター）
- カート機能（追加・削除・数量変更）
- 注文フォーム（銀行振込・請求書払い → メール通知）
- クレジットカード・Apple Pay（Stripe Checkout）
- お問い合わせフォーム（自動返信 + 管理者通知）
