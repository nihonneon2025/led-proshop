「屋外でも使えると書いてあったのに水が入って故障した」「IP65を選んだが水没環境では機能しなかった」——LEDテープのIP（防塵・防水）規格は番号ひとつで守られる状況が全く変わります。本記事では防水規格の正確な読み方から、設置場所別の最適選定、接続部・端末部の防水処理まで実務に直結する情報を整理します。

1. IP規格（IEC 60529）の正しい読み方

IP表記の構造
IP 65
第1特性数字（防塵等級 0〜6）
6 = 粉塵の侵入ゼロ（完全防塵）
5 = 粉塵侵入を有害なほど許さない
0 = 保護なし
第2特性数字（防水等級 0〜9K）
8 = 水没（連続使用可）
7 = 一時的な水没
6 = 強力な水流・ジェット
5 = あらゆる方向からの水流
4 = 全方向からの飛沫
3 = 降雨・噴霧

LEDテープで一般的なIP等級の比較

IP20 防水なし（屋内専用）
保護内容：防塵・防水なし
乾燥した屋内のみ使用可能。コーニス照明・天井間接照明など水がかからない場所向け。コストが最も低く、放熱性も良好。
IP44 飛沫防水
保護内容：1mm以上の固体侵入保護 + 全方向飛沫
軒下・洗面所・トイレなど直接水流がかからない水回り向け。浴室や厨房には不十分。エントランス屋根下などに採用。
IP65 防塵・防噴流水 屋外標準
保護内容：完全防塵 + あらゆる方向からの水流
屋外軒下・サイン照明・屋根付き外構など最も一般的な屋外用途。雨に濡れる場所でも使用可能。ただし水没はNG。
IP67 防塵・一時水没
保護内容：完全防塵 + 最大1m水深に30分耐える
噴水縁・池の縁・水飛沫が直接かかる場所・路面埋込みライト向け。水中照明は水深と圧力を確認してからIP67かIP68かを選択。
IP68 防塵・連続水没
保護内容：完全防塵 + 規定の深さ・時間での連続水没
プール水中・観賞魚水槽・連続浸漬が必要な用途。「IP68(3m/30min)」のように条件が製品仕様に明記されているため確認必須。
⚠️ よくある誤解：IP65 = 屋外どこでもOKではない
  • IP65は「水流への耐性」であり、長時間の水没には対応していません
  • 路面に埋め込む用途（車が踏む場所）は防圧力基準が別途あります（IK規格参照）
  • 塩害地域（海岸近く）ではIP65でも腐食が進む場合があります。塩水対応コーティングの有無を確認
  • UV劣化：屋外では5年以上の直射UV暴露でシリコンコーティングが劣化します。UV安定剤入りか確認

2. 設置場所別・推奨IP等級一覧

設置場所 水の状況 最低推奨IP 補足
屋内・乾燥した場所
（コーニス・天井・棚下）		 水なし IP20 放熱性を重視。IP65以上はシリコン被覆で放熱が下がるため必要なし
軒下（屋根あり）
降雨が直接当たらない		 飛沫・湿気あり IP44 強風時に横殴りの雨が当たる場合はIP65推奨
屋外露出部
（看板側面・外壁・フェンス）		 直接降雨あり IP65 推奨 シリコンコーティング品。UV安定剤の有無を確認
洗面所・トイレ 飛沫程度 IP44 洗面ミラー上・間接照明に。シャワースペース側は IP65
浴室（浴槽外） 高湿度・湯気・飛沫 IP65 推奨 日本の建築基準法では浴室用照明器具の防湿形対応が必要
厨房・食洗機周辺 水流・蒸気あり IP65 推奨 油煙も多いため清掃しやすいシリコンコーティング品が好ましい
プール縁・噴水周辺 常時水飛沫 IP67 水面下への浸漬がある場合はIP68。塩素対応も確認
水中（プール・水槽） 常時水中 IP68 推奨 仕様書に「水深×時間」の数値明記があるか確認必須
冷蔵・冷凍ショーケース 結露・低温 IP44〜IP65 結露水が落ちる場合はIP65。低温での動作保証温度（−20℃等）を確認

3. LEDテープの防水コーティング構造を知る

防水の仕組み：4つの構造タイプ

タイプ 構造 IP等級 特徴
ノンコート 基板むき出し IP20 コスト最安・放熱最良。屋内専用。切断・はんだ付けが容易
エポキシコート 基板面に薄い樹脂塗布 IP44〜IP54 飛沫に耐えるが水流・水没には不向き。切断面が無防水になる
シリコンコーティング
（半透明シリコン被覆）		 テープ全体をシリコンで包む IP65 屋外標準。柔軟性あり。切断後に切断面の防水処理が必要
シリコンチューブ
（チューブ内に封入）		 シリコン管内に基板を封入 IP67〜IP68 完全防水。コストが高い。熱が逃げにくいため放熱設計に注意
切断面の防水処理は必須：IP65以上のシリコンコーティング品でも、切断した端末部分は防水処理が必要です。シリコン充填剤（シリコーンシーラント）を端末に充填し、24時間以上乾燥させてから設置してください。コネクタ接続箇所も同様に防水コネクタまたはシリコン充填で処理します。

4. 接続部・端末部の防水処理手順

LEDテープの防水施工で最も失敗が多いのは接続箇所と端末の処理です。テープ本体がIP65でも接続部が無防水では水が浸入します。

  • 防水コネクタを使用する：屋外・水回り施工では必ず防水コネクタ（IP65対応）を使用。一般的な押し込みコネクタは防水性なし。防水コネクタはゴムパッキンまたはシリコンパッキン内蔵のものを選択。
  • 端末処理：シリコーンシーラントで充填：テープの切断端部にシリコーンシーラント（中性硬化型）を充填し、表面が丸くなるまで覆う。硬化前に余分を拭き取り、24時間以上養生後に設置。
  • 電源ケーブルの引き込み部に防水グランドを使用：PSU（電源）へのケーブル引き込み口には防水グランド（ケーブルグランド）を取り付け、隙間をなくす。
  • コネクタ接続後に自己融着テープを巻く：コネクタ部分全体を自己融着テープ（ブチルテープ）で二重に巻き付け、防水コネクタの補強処理とする。特に水中・水飛沫が直接当たる場所では必須。
  • 施工後の通電試験前に防水確認：外観での目視確認後、霧吹きで水をかけて30分後に点灯テスト。水分が乾かないうちに通電すると短絡リスクがあるため、十分乾燥させてから通電。

5. 設置場所から最適IP等級を選ぶ5問フロー

  1. 設置場所は屋内ですか？屋外ですか？ → 屋内（水が絶対かからない）: IP20で十分 ／ 屋外または水回り: 次の質問へ
  2. 水が直接かかりますか？（降雨・水流・シャワー等） → 飛沫程度（軒下・洗面）: IP44 ／ 直接かかる（屋外露出・厨房・浴室）: IP65以上
  3. 水中に沈む可能性がありますか？（噴水・プール縁・路面埋込） → 一時的（たまに水が溜まる）: IP67 ／ 常時水中（プール内・水槽内）: IP68
  4. 塩害環境（海から1km以内）・UV暴露が強い南面ですか？ → 該当: IP65以上 + UV安定剤入り + 塩水対応コーティング品を選択
  5. 調光（PWM）を使いますか？シリコンチューブ品（IP67/68）でも調光可能ですか？ → 対応確認: IP67/68のシリコンチューブ品は製品ごとに調光対応の有無を仕様書で確認。対応していない場合はIP65のシリコンコーティング品で代替

6. 防水等級が高いほど放熱に不利になる理由と対策

防水コーティングはLEDチップを水から守る一方で、熱の放出経路を塞ぐという副作用があります。

防水等級と放熱効率の関係
IP等級 コーティング 放熱効率 対策
IP20 なし ◎ 最高 標準アルミチャンネルで十分
IP44 薄いエポキシ ○ 良好 大きめチャンネルを推奨
IP65 シリコン薄膜 △ やや低下 同W/mなら1サイズ上のチャンネル、または W/mを1段階下げる
IP67/68 シリコンチューブ ✗ 大幅に低下 W/mを6〜8W/mに抑制。水中ならば水冷効果で補える
実践アドバイス：屋外でIP65のテープを10W/m以上で使う場合、チャンネルのサイズを20mm Tチャンネル以上に引き上げてください。シリコン被覆が熱の放散を妨げるため、チャンネルを大きくして補完します。プールや水中用のIP68品は水自体が冷媒として機能するため、水中では放熱が改善します（ただし電源・接続部は水中に入れないこと）。
