施工背景と課題

関東圏のセルフガソリンスタンド（敷地面積約1,200㎡・建屋800㎡）を運営するオーナー様より、「24時間営業の電気代が毎月25万円近くかかっており、他店との競争で価格を下げられない」とのご相談をいただきました。現地調査の結果、キャノピー照明にメタルハライドランプ（400W×8台）を使用し続けており、これが全体消費電力の約30%を占める最大の電力損失源であることが判明しました。

加えて洗車ライン・コンビニ売店・事務所の蛍光灯が経年劣化により光束が著しく低下しており、お客様から「暗い」「汚く見える」との声も上がっていました。今回のLED全面換装は省エネと集客力回復の二兎を同時に獲得する施策として実施しました。

ガソリンスタンドLEDの技術ポイント

🌧️
全天候IP65・IP67
屋外キャノピーは雨・結露・温度変化（-10〜+50℃）に常時さらされる。IP65以上で水噴射に耐え、洗車ラインはIP67（水没30分）対応のシリコン封止型を採用。
⚡
MH再点灯問題の解消
旧メタルハライドは停電・球切れ後の再点灯に30分かかり、夜間停電時にキャノピー全暗になるリスクがあった。LEDは瞬時点灯・瞬時復帰で安全性を確保。
🌙
深夜PIR省エネ制御
深夜0〜5時の給油台数は日中の10〜15%。PIRセンサーで20%待機に自動調光し、客検知で0.3秒以内に100%復帰。深夜の無駄な全点灯をゼロに。
🚗
路上視認性と集客向上
キャノピー600lx均一・5000K鮮明な白色光は、旧MH（黄色みのある演色）より道路からの視認性が大幅に向上。夜間の入店率改善効果が期待できる。
🔩
防振マウント設計
給油車両の走行振動・強風・キャノピーのたわみに対し、アルミチャンネルを防振クリップ5点固定。電源ユニットも防振ゴム台座付きボックスに収容。
🍎
コンビニRa85食品演色
弁当・惣菜・鮮度感のある飲料がRa85の演色で本来の色味に見える。旧蛍光灯Ra72との差により食品の売場が格段に明るく鮮やかになり購買促進に貢献。
【防爆エリアについての注意事項】
給油機直近（IEC 60079規格による爆発危険エリアZone 1/Zone 2）に照明器具を設置する場合は、防爆認定（Ex認定）取得済みの専用LED器具が必要です。本施工事例のキャノピーテープ照明はZone 2外縁（距離・高さで危険区域外）への設置を前提としており、防爆認定LEDの適用対象外エリアに限定しています。防爆エリアへのLED導入は、専門の防爆設備業者および所轄消防署への事前確認が必須です。

使用製品・仕様

今回採用したLED製品ラインナップ

【Zone A キャノピー】高輝度COBテープ 5000K / Ra80 / 10W/m — IP65対応シリコン封止型・200m敷設。アルミチャンネル（60×15mm）防振固定マウント一体型。
【Zone B 洗車ライン】防水COBテープ 5000K / Ra80 / 10W/m — IP67完全防水・シリコン封止全周充填型・150m敷設。耐薬品性シリコンシース採用。
【Zone C コンビニ】高演色COBテープ 5000K / Ra85 / 10W/m — 食品陳列対応・120m敷設＋棚下補助ライン追加。
【Zone D 事務室】標準COBテープ 4000K / Ra85 / 10W/m — 30m敷設。PIR連動PWM調光コントローラー付属。
【電源ユニット】DC24V スイッチング電源（IP54防水ボックス収容）— フリッカー0%・PFC搭載・過負荷保護。Zone別独立電源回路で一部故障の全域伝播を防止。
【センサー制御】PIR人感センサー（応答0.3秒）— Zone A/B/D に設置。タイムスケジュール併用（深夜帯自動モード切替）。
