LEDテープ 12V vs 24V 選び方｜業務用施工の判断基準

執筆: LED PRO SHOP 編集部 公開日: 2026-04-30 更新日: 2026-04-30

目次

    「12V と 24V、どちらを選べばいいか」は、LEDテープを業務で扱う電気工事業者・看板施工業者が必ず一度はぶつかる問いです。家庭用 DIY であれば「とりあえず 12V」で済むことも多いものの、業務用の長尺配線・複数台同時点灯・什器の分岐多数といった条件下では、電圧の選定ミスが現場でのちらつき・電圧降下・PSU 容量不足・追加配線コストの形で跳ね返ってきます。本記事は、業務施工の現場目線で 12V と 24V の違いを整理し、看板・什器・ファサード別の判断基準と、配線長別の電圧降下早見表、5問で答えが出る判断フローチャートまでを一気通貫で示します。

    12V と 24V の基本的な違い

    LEDテープの世界では、直流 12V 仕様と直流 24V 仕様が業務用主流の2系統です。同じ COB 構造・同じ発光色温度でも、駆動電圧が違えば、配線長の上限・PSU の選び方・分岐設計の柔軟性が大きく変わります。

    • 短尺・分岐多数の什器照明 → 12V 有利
    • 長尺・単純配線の看板/ファサード照明 → 24V 有利
    • どちらでも成立するゾーンも存在する（配線長 5〜10m・1電源あたり 200W 級など）

    電圧降下と長尺配線の関係

    LEDテープを長く引き回すほど、ケーブルや基板パターンの抵抗で電圧が落ちます。これが電圧降下で、奥側のテープが暗くなる・色がずれる・点滅するといった現場トラブルの主因です。同じ消費電力（W）でも 24V のほうが電流（A）が半分になるため、電圧降下も小さくなります。

    配線長別 電圧降下早見表（業務用 LEDテープ・1本給電想定）

    配線長12V 系（10W/m）想定24V 系（10W/m）想定実務上の判断
    5m約 2〜4% 降下約 1〜2% 降下どちらも実用域
    10m約 5〜8% 降下約 2〜4% 降下12V は両端給電か中間給電を検討
    15m約 8〜12% 降下約 4〜6% 降下12V は単一電源では推奨しない
    20m約 12〜18% 降下約 6〜9% 降下20m は 24V 一択＋給電工夫が定石

    上記は業務用 COB / SMD テープを 1本連続給電した場合の目安です。実際にはケーブル断面積・接触抵抗・パターン抵抗で変わるため、現場では 5〜10% を許容上限とした余裕設計を推奨します。

    電源容量と費用感

    観点12V 系24V 系
    同じ W 数での電流多い（2倍）少ない（1/2）
    同じ長さでの電圧降下大きい小さい
    必要ケーブル断面積太め細めで OK
    業務用主流容量帯200W〜400W（短〜中尺向け）200W〜400W（中〜長尺向け）
    業務長尺施工の親和性中高

    業務用施工で 12V を選ぶケース

    短尺・分岐多数の什器照明

    1. 小ロット・狭小スペースの取り回し：1什器あたりの消費電力が小さく、什器内蔵・棚下・サイン背面など狭小スペースに PSU を収めやすい。
    2. 分岐設計が柔軟：1電源から複数の什器への並列分岐がしやすく、各分岐が短尺なので電圧降下を気にしなくて良い。
    3. 既設配線との互換：既存の DC12V 配線（ハロゲン置き換え・古い LED 看板の更新など）に乗せ替える場面で、追加工事を最小化できる。

    12V 系電源の選定ポイント

    • PSE 取得済みか：店舗・事務所・什器に常設する電源は PSE 済みを必須条件に。
    • 負荷率 70% 設計：定格 200W の PSU には実負荷 140W までで運用するのが鉄則。
    • 力率改善（PFC）の有無：複数 PSU を同一系統に並べる現場では、PFC 付きでないとブレーカー容量を圧迫します。
    • 保護回路：過電流（OCP）・過電圧（OVP）・過熱（OTP）・短絡（SCP）の4点が揃っているか。
    • ケース仕様：屋外/半屋外なら IP67 クラスを選定。

    PSU 容量計算は、関連記事「LEDテープ用 PSU 容量計算 早見表」で詳述しています。

    業務用施工で 24V を選ぶケース

    長尺・看板・ファサード

    1. 電圧降下が半分以下：同じ 10m 配線でも 24V のほうがムラが出にくい。
    2. 大容量 PSU が主流：200W〜400W クラスの 24V PSU は業務用ラインナップが厚く、長尺一括給電に向く。
    3. テープ自体のラインナップが充実：高 CRI・高効率・防水仕様などプロ向け 24V テープの選択肢が広い。
    4. 配線材コストの低減：電流が半分になるため、長距離配線でもケーブル断面積を抑えられ、施工工数も減る。

    24V 系電源の選定ポイント

    • 連続定格 vs ピーク定格：連続定格 W 数で判断。
    • 複数並列時の電流バランス：Active Current Sharing 機能または個別給電設計を選択。
    • 屋外設置の防水・防塵：雨水侵入の可能性がある場所は IP67 以上。
    • 調光対応の有無：PWM 調光・0-10V 調光・DALI など、調光プロトコルとテープ側コントローラの整合を事前確認。
    • 入力電圧範囲：AC200V 三相が来ている現場では AC180-264V 対応を選ぶと使い回しが効きます。

    よくある質問（FAQ）

    Q1. 12V と 24V を1つの現場で混在させても問題ないですか？

    技術的には可能です。ただし電源系統と制御信号系統を完全に分離する必要があり、施工後の保守を考えると、1現場 1電圧に統一するのを推奨します。

    Q2. 12V のテープを 24V PSU で点灯させたら明るくなりますか？

    絶対にやめてください。即時破損または火災の原因になります。電圧表記は必ずテープ仕様と PSU 仕様を完全一致させてください。

    Q3. 配線長 8m の現場、12V でいけますか？

    「いける」境界です。配線断面積 1.25sq 以上・両端給電または中間給電・テープ消費電力が 8W/m 以下、の3条件が揃えば 12V でも実用域に収まります。ただし追加工事費を考えると 24V を最初から選ぶほうが総コストで有利になることが多いです。

    Q4. 看板の文字内蔵で短尺・複数チャンネルの場合は？

    12V が定石です。各文字が数 cm〜30cm 程度の短尺なので電圧降下を気にする必要がありません。

    Q5. 後から長尺に拡張する可能性がある案件は？

    最初から 24V で組むのが安全です。「将来 5m 以上拡張する可能性がある」と聞いた時点で 24V を提案するのがプロの判断です。

    まとめ — 次のアクション
